# 🍜 美食大转盘

> 手机可访问的美食随机选择器，转动转盘决定今天吃什么！

## 📖 使用说明

### 第一步：选择分类

点击上方的分类按钮，选择你想吃的类型：

| 分类 | 内容 |
|------|------|
| 🍱 午餐 | 黄焖鸡、麻辣烫、兰州拉面、炒饭、螺蛳粉等 10 种 |
| 🍽️ 晚餐 | 火锅、烧烤、小龙虾、烤鱼、日料等 10 种 |
| 🧋 饮品 | 奶茶、咖啡、柠檬水、椰子水等 8 种 |
| 🍿 零食 | 薯片、冰淇淋、糖葫芦、坚果等 8 种 |
| ✏️ 自定义 | 添加自己喜欢的选项，最多 12 个 |

### 第二步：转动转盘

点击金色中心按钮，或直接点击转盘，转盘就会开始旋转 🎯

### 第三步：揭晓答案

转盘停止后，指针指着的那一项就是今天的答案！🎉

## ✨ 功能特点

- 🍱 **4 大分类** — 午餐、晚餐、饮品、零食，共 36 种美食
- ✏️ **自定义模式** — 添加自己喜欢吃的选项
- 📱 **手机适配** — 完美支持手机浏览器，触屏操作流畅
- 🎆 **烟花特效** — 每次转出结果都有烟花庆祝
- 📜 **历史记录** — 记录今天转过的所有结果
- 🔄 **永久免费** — 基于 GitHub Pages 托管，无需任何费用

## 📝 自定义美食列表

如果你想添加自己的美食选项，可以修改 `index.html` 中 `categories` 对象里的 `custom` 数组：

```javascript
custom: {
  items: [
    { name: '你的美食', emoji: '🍕', color: '#FF6B35' },
    // 添加更多...
  ]
}
```
